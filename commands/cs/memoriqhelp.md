**Memoriq v4 — Aktivni**

Tvuj AI ma trvalou pamet. Fakty, rozhodnuti a kontext preziji mezi sessions.

**Prikazy:**

| Prikaz | Co dela |
|--------|---------|
| `/status` | Statistiky pameti a zdravi projektu |
| `/recall [dotaz]` | Hledani v pameti |
| `/harvest` | Extrakce a ulozeni znalosti z aktualni session |
| `/onboard` | Proskenovani projektu a vytvoreni zakladni pameti |
| `/onboard-all` | Hromadny onboarding vsech projektu |
| `/forget [dotaz]` | Smazani konkretnich faktu |
| `/identity` | Sprava Identity Card pro deploy |
| `/consolidate` | Organizace pameti — clustery, tiers, kontradikce |
| `/tui` | Spusteni vizualniho TUI dashboardu |
| `/memoriqhelp` | Tato napoveda |

**Automaticke chovani** (neni treba nic delat):
- Session bridge — navaze kde minula session skoncila
- Proaktivni pamet — Claude uklada dulezite fakty automaticky
- Crash recovery — obnovi kontext pokud session spadne
- Detekce zastarani — varuje kdyz se zdrojovy soubor zmenil

**TUI Dashboard:** Spust `memoriq` v jakemkoliv terminalu.
